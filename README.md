# ARIA: Automated Remote Interactive Assistant 🤖

**ARIA** is an intelligent clinical coordination engine designed to automate the administrative workflow of a modern health facility.

## 🌟 The Core Mission
Administrative friction—missed appointments and manual scheduling—costs clinics time and reduces patient outcomes. ARIA was developed to automate these touchpoints using high-veracity data sync between n8n and professional databases.

## 🚀 Key Features
- **Smart Scheduling & Reminders:** Automates 24-hour and 1-hour pre-appointment notifications via Telegram and SMS.
- **Dynamic Response Handling:** Processes patient confirmations (Yes/No/Reschedule) and updates the clinic database in real-time without human intervention.
- **Administrative Guardrails:** Automatically flags scheduling conflicts and empty slots for the clinical team.
- **Interactive Assistance:** Provides patients with instant logistics info (Clinic location, preparation instructions).

## 🛠️ Tech Stack
- **Engine:** n8n (Advanced Workflow Orchestration)
- **Database:** Airtable (Relational Clinical Records)
- **Communication:** Telegram Bot API / Twilio (SMS)
- **Programming:** JavaScript (Functional Logic Nodes)
